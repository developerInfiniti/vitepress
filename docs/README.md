# Документация IT шпаргалок

Этот проект представляет собой многоязычную документацию IT шпаргалок, созданную с использованием VitePress.

## Поддерживаемые языки

В настоящее время документация доступна на следующих языках:

- Русский (основной язык)
- Украинский

## Структура проекта

```
├── docs/                  # Основная директория с документацией
│   ├── .vitepress/       # Конфигурация VitePress
│   ├── ...               # Документация на русском языке
│   └── uk/               # Документация на украинском языке
```

## Добавление новых документов

### Для русской версии

Добавляйте новые документы непосредственно в соответствующие директории в корне `docs/`.

### Для украинской версии

Добавляйте новые документы в соответствующие директории внутри `docs/uk/`.

## Перевод существующих документов

Для перевода существующего документа на украинский язык:

1. Найдите оригинальный документ в корне `docs/`
2. Создайте соответствующую директорию в `docs/uk/` (если она еще не существует)
3. Создайте файл с тем же именем в этой директории
4. Переведите содержимое документа на украинский язык

## Навигация между языками

Переключение между языками доступно в верхней части сайта через выпадающее меню языков.
