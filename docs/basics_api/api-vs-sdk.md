# API vs SDK

## 🔍 Вступ

У розробці часто використовуються поняття **API** та **SDK**, і хоча вони тісно пов’язані, мають суттєві відмінності.

---

## 📘 Що таке API?

**API (Application Programming Interface)** — це набір правил, який дозволяє одній програмі взаємодіяти з іншою. API визначає, **як програма може запитувати і надсилати дані** іншій системі.

### 🔹 Приклад:
Вебсайт використовує **Google Maps API**, щоб відображати карту з адресою.

```js
// Виклик Google Maps API
const map = new google.maps.Map(document.getElementById("map"), {
  center: { lat: 50.4501, lng: 30.5234 },
  zoom: 12,
