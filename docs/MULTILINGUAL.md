# Руководство по многоязычности

## Обзор

Проект поддерживает два языка:
- Русский (основной язык, расположен в корне директории `docs/`)
- Украинский (расположен в директории `docs/uk/`)

## Структура проекта

```
├── docs/                  # Основная директория с документацией
│   ├── .vitepress/       # Конфигурация VitePress
│   │   └── config.mts    # Файл конфигурации с настройками многоязычности
│   ├── ...               # Документация на русском языке
│   └── uk/               # Документация на украинском языке
│       ├── basic_docker/ # Украинская версия документации по Docker
│       ├── basics_vue/   # Украинская версия документации по Vue.js
│       └── ...           # Другие разделы на украинском языке
```

## Добавление нового контента

### Для русской версии

1. Создайте новый Markdown-файл в соответствующей директории в корне `docs/`
2. Добавьте ссылку на новый документ в боковую панель в файле `docs/.vitepress/config.mts` (если необходимо)

### Для украинской версии

1. Создайте такую же структуру директорий в `docs/uk/`, как и для русской версии
2. Создайте новый Markdown-файл с тем же именем, что и в русской версии
3. Переведите содержимое на украинский язык

## Перевод существующего контента

Для перевода существующего документа на украинский язык:

1. Найдите оригинальный документ в корне `docs/`
2. Создайте соответствующую директорию в `docs/uk/` (если она еще не существует)
3. Создайте файл с тем же именем в этой директории
4. Переведите содержимое документа на украинский язык

## Конфигурация многоязычности

Настройки многоязычности находятся в файле `docs/.vitepress/config.mts`. Основные разделы:

```js
locales: {
  root: {
    label: 'Русский',
    lang: 'ru',
    // Конфигурация для русского языка
  },
  uk: {
    label: 'Українська',
    lang: 'uk',
    // Конфигурация для украинского языка
  }
}
```

## Переключение языков

Переключение между языками доступно через выпадающее меню в верхней части сайта. Конфигурация переключателя языков:

```js
themeConfig: {
  langMenuItems: [
    { text: 'Русский', link: '/' },
    { text: 'Українська', link: '/uk/' }
  ],
  // Другие настройки темы
}
```

## Рекомендации по переводу

1. Сохраняйте структуру оригинального документа
2. Переводите все текстовые элементы, включая заголовки, описания и комментарии в коде
3. Не переводите имена переменных, функций и другие технические термины в примерах кода
4. Сохраняйте форматирование Markdown и HTML-теги
5. Обновляйте ссылки, чтобы они указывали на соответствующие страницы на украинском языке
