# Українська версія IT шпаргалок

Цей каталог містить українську версію документації IT шпаргалок.

## Структура каталогів

Структура каталогів української версії повинна відповідати структурі російської версії:

```
├── uk/                    # Основний каталог української версії
│   ├── basic_docker/      # Документація по Docker
│   ├── basics_vue/        # Документація по Vue.js
│   ├── basics_react/      # Документація по React
│   └── ...                # Інші розділи документації
```

## Додавання нових документів

Для додавання нового документа українською мовою:

1. Створіть відповідний каталог в директорії `uk/` (якщо він ще не існує)
2. Створіть Markdown-файл з тим же ім'ям, що і в російській версії
3. Перекладіть вміст документа українською мовою

## Навігація

Навігація для української версії налаштована в файлі конфігурації VitePress. При необхідності внесення змін у навігацію, відредагуйте файл `docs/.vitepress/config.mts`.
