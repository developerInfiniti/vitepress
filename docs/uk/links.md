# Корисні посилання

На цій сторінці зібрані корисні посилання на ресурси для розробників.

## Документація

- [MDN Web Docs](https://developer.mozilla.org/) - Всеосяжна документація з веб-технологій
- [Vue.js](https://vuejs.org/) - Офіційна документація Vue.js
- [React](https://reactjs.org/) - Офіційна документація React
- [TypeScript](https://www.typescriptlang.org/docs/) - Офіційна документація TypeScript
- [Node.js](https://nodejs.org/en/docs/) - Офіційна документація Node.js

## Навчальні ресурси

- [freeCodeCamp](https://www.freecodecamp.org/) - Безкоштовні курси з програмування
- [Codecademy](https://www.codecademy.com/) - Інтерактивні курси з програмування
- [Udemy](https://www.udemy.com/) - Онлайн-курси з різних технологій
- [Coursera](https://www.coursera.org/) - Онлайн-курси від провідних університетів

## Інструменти

- [GitHub](https://github.com/) - Платформа для хостингу коду та спільної розробки
- [CodePen](https://codepen.io/) - Онлайн-редактор для фронтенд-розробки
- [VS Code](https://code.visualstudio.com/) - Популярний редактор коду
- [GitLab](https://gitlab.com/) - Альтернативна платформа для хостингу коду

## Спільноти

- [Stack Overflow](https://stackoverflow.com/) - Питання та відповіді для програмістів
- [Dev.to](https://dev.to/) - Спільнота розробників
- [Reddit](https://www.reddit.com/) - Різноманітні підфоруми для розробників
- [Hashnode](https://hashnode.com/) - Платформа для блогів розробників

## Блоги та новини

- [CSS-Tricks](https://css-tricks.com/) - Статті та поради з CSS
- [Smashing Magazine](https://www.smashingmagazine.com/) - Статті про веб-дизайн та розробку
- [A List Apart](https://alistapart.com/) - Статті про веб-стандарти та найкращі практики
- [JavaScript Weekly](https://javascriptweekly.com/) - Щотижневий дайджест новин JavaScript
