# Настройка и запуск многоязычного проекта

## Установка

Для работы с многоязычной версией проекта выполните стандартную установку:

```bash
npm install
# или
yarn install
```

## Запуск сервера разработки

Запустите сервер разработки стандартной командой:

```bash
npm run docs:dev
# или
yarn docs:dev
```

После запуска сервер будет доступен по адресу `http://localhost:5173`

## Проверка многоязычности

1. Откройте сайт в браузере
2. В верхней части сайта найдите переключатель языков
3. Переключитесь между русской и украинской версиями сайта

## Сборка для продакшена

Сборка многоязычного проекта выполняется стандартной командой:

```bash
npm run docs:build
# или
yarn docs:build
```

Сборка будет содержать все языковые версии сайта.

## Предварительный просмотр сборки

```bash
npm run docs:preview
# или
yarn docs:preview
```

## Добавление нового языка

Если в будущем потребуется добавить поддержку дополнительных языков:

1. Добавьте новую локаль в секцию `locales` в файле `docs/.vitepress/config.mts`
2. Добавьте новый пункт в меню переключения языков в секции `langMenuItems`
3. Создайте соответствующую директорию для нового языка в `docs/`
4. Скопируйте структуру директорий из русской версии
5. Переведите содержимое на новый язык

Пример добавления английского языка:

```js
locales: {
  root: {
    label: 'Русский',
    lang: 'ru',
    // ...
  },
  uk: {
    label: 'Українська',
    lang: 'uk',
    // ...
  },
  en: {
    label: 'English',
    lang: 'en',
    title: "IT Cheatsheets",
    description: "Your quick reference guide for developers"
    // ...
  }
}
```

И добавление в меню языков:

```js
langMenuItems: [
  { text: 'Русский', link: '/' },
  { text: 'Українська', link: '/uk/' },
  { text: 'English', link: '/en/' }
]
```
